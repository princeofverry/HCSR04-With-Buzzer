
# Arduino HCSR04 With Buzzer

This is prototype sensor parking


## Authors

- [@princeofverry](https://www.github.com/princeofverry)


## Tools Stack

* Arduino Uno R3
* Cable Jumper
* BreadBoard
* HCSR04 aka Ultrasonic Sensor
* Buzzer

## PIN
* Trigger = Digital 4
* Echo = Digital 3
* Buzzer = Digital 2
